# Netflix Style Catalog 🎬

Project developed during the **Alura Front-End with AI Immersion**.

The goal of this project was to create an interface inspired by streaming platforms, with a movie catalog organized into categories, simulating the browsing experience of services like Netflix.

## 🚀 Features

- Catalog organized by categories  
- Movie cards with cover images  
- YouTube trailer integration  
- Interface inspired by streaming platforms  
- Modular JavaScript structure  

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript  
- YouTube video integration  

## 📚 What I Learned

During the development of this project, I practiced concepts such as:

- Layout structuring  
- Working with data in JavaScript  
- Organizing interface components  
- Using external assets (images and trailers)

## 🎓 Context

This project was developed as part of the **Alura Front-End with AI Immersion**, focused on practicing interface development concepts and using modern tools in the creation process.
